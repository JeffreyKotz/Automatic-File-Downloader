# Automatic-File-Downloader
**By Jeffrey Kotz**

## Description
This automatic file downloader can receive a list of links to download automatically

## Features
+ Program downloads files from user specified URLs automatically.
    + Downloaded files are saved to directory specified by user.
+ All download operations occur asynchronously and may be canceled by the user at any point during the download.
+ Program can process any number of given urls dynamically adding them to the existing URLs.
